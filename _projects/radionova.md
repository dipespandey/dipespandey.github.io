[Radionova](https://radionova.azurewebsites.net) is a CMS for online radio station of University of Oslo, Norway.
It is built using the Wagtail CMS, and is used by the radio station to manage their content. Operators are able to write 
articles, book rooms, and manage other content using the CMS.

It is hosted in Azure App Services, and uses PostgreSQL as the database.
