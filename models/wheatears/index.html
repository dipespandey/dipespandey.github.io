Wheatears counting in images

Counting wheatears from images using deep learning

You can find more details in the paper and the colab notebook below.
- The paper
- Colab URL

We are supplied with a raw set of images of a wheatfield with a flock of wheatears. We want to count the number of wheatears in each image. This is an actual problem that requires a lot of manual work from farmers. Apparently, wheatears count is a great indicator of the health of the wheat field.

To accomplish this, we used two algorithms:

  1. Faster R-CNN
  2. EfficientDet

While EfficientDet is a better model than Faster R-CNN, we wanted to compare the two models to see how they perform on this task.

Dataset: We used the Global Wheat Dataset for training and testing the models.
